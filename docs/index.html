Everything you need

Built for real classrooms.
Powered by real AI.

🎥
Live Face Detection
OpenCV Haar Cascade detects faces in real-time at 30fps from any webcam, even in varied lighting.
🧠
KNN Recognition
Scikit-learn KNN model trained on 50 face samples per user. Instant identification with zero cloud dependency.
⚡
Real-Time WebSocket
Dashboard updates live the moment attendance is marked — no page refresh needed, powered by Flask-SocketIO.
🗄️
SQLite Database
Durable SQLite backend replaces CSV with structured querying, history, and UNIQUE constraints per day.
📊
REST API
Full JSON API: GET today's records, history by date, user list, stats — integrate with any system.
📁
Export: CSV + Excel
One-click export to CSV or XLSX with all attendance data, perfect for reports and institutional records.
👤
Admin Panel
Full user management: view all registered users, delete users, see attendance history, and export reports.
🐳
Docker Ready
One command deployment with Docker Compose. Zero environment setup. Works everywhere Python runs.
🔄
GitHub Actions CI/CD
Automated testing on every push. Linting, import checks, and deployment pipeline baked in from day one.
